# WriterPod Studio

A comprehensive content creation platform for writers with AI-powered text-to-speech, chapter-wise publishing, and social sharing capabilities.

## Features

- **Chapter-wise Story Publishing**: Create serialized content with multiple chapters
- **AI Text-to-Speech**: Convert written content to natural-sounding audio
- **AI Writing Assistant**: Get writing suggestions and grammar corrections
- **Background Music & Effects**: Add atmospheric audio to enhance stories
- **Analytics Dashboard**: Track story performance and engagement
- **Social Media Sharing**: Share content across platforms
- **Creator Community**: Follow and connect with other creators
- **Monetization**: Earn through creator programs
- **Substack-like Publications**: Create and manage newsletters
- **Microblogging (Notes)**: Share short-form posts like Twitter
- **Community Chats**: Private discussions for subscribers
- **Multimedia Posts**: Support for text, audio, and video content

## Quick Start

### Backend Setup
```bash
cd backend
npm install
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

## Technology Stack

**Backend:**
- Node.js & Express
- MongoDB with Mongoose
- JWT Authentication
- Text-to-Speech API integration
- File upload handling

**Frontend:**
- React 18
- React Router for navigation
- Axios for API calls
- TailwindCSS for styling
- Rich text editor for content creation
